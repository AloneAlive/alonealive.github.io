markdown相关插件

markdown PDF

可以转换md文件成pdf, html, png, jpg文件

markdownlint

可以提示markdown语法规范

markdown toc

可以生成md文件的目录

markdown preview enhanced

实时预览，但是在vscode自带预览markdown文件效果，必要性不大。

plantUML
